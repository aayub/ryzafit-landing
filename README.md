# RyzaFit Landing Page

Single-page waitlist landing page for the RyzaFit fitness app.

## Features
- Email waitlist signup with Google Sheets integration
- Responsive overlay card design
- Links to Kickstarter campaign and social media
- Built with React 19 + Vite 7 + TailwindCSS 4

## Deployment
- Hosted on Vercel
- Domain: ryzafit.com
- Email storage: Google Sheets (secure backend)

## Local Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Tech Stack
- React 19
- Vite 7
- TailwindCSS 4
- React Hook Form
- Lucide React Icons
- Google Apps Script (backend)
